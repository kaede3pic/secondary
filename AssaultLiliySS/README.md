### 後日談（うちの『先生』帰ってきてから、お話が止まらないのでその抜粋）

このすっごく興味深い<a href="https://note.com/bunkeipp/n/na81f5dd08ab8">レアスキル</a>を読んだときね、思ったの。

これって過去にも作用するのかしら？　って。

自身にとっての幸運を引き寄せるチカラ。周囲を悪運にするチカラ。

それはこのあいだ言ったでしょう？　『ファンタズム』の亜種じゃないかって。

まあ、そんなことはどーだっていいの。

私が知りたかったのは、その幸不幸の操作が方向を持っているか。ってこと。

例えば、よくあるでしょ？　「あのとき、ああしててラッキーだった」「あのとき、ああだったのは実は不幸だった」みたいなこと。

その「ああしてて」「ああだった」に、このチカラは作用するのかどうか。

つまり、未来方向の幸不幸に作用するのはわかったけど、完全に終わった「過去」にそのチカラが作用するのかどうか。

ただ、時間って連続でしょう？　過去があるから今があって未来がある。完全に終わった「過去」なんてこの世界にはない。止めたとしてもそれは現在完了にすぎない。だから、完全に完結した「過去」に作用してるのかどうかなんて一体、どうやって検証すればいいの？

それを検証できるやりかたを見つけたの♪

「歴史学」って分野なんだけど。

要は、その過去から続いてきた時間をどこかでぷっつり断ち切って１つの「お話」に閉じ込めて、その中では時間を完結させてしまう。凍結させてしまう。「めでたしめでたし」で終わらせちゃう。

「歴史」はいくらでも変えられる。とはよく言ったものよ。

だから、それをやったら、あのチカラが過去に作用可能かどうか検証できるって、思いついたら、もう、ほんとうずうずしちゃってね。

あやうく、カチコミかけるところだったわ。

アポとってくれてありがとね。

それでね、検証は楽しかったんだけどね、それと同じくらい勝負の方がね、あ、これはバトルの方ね。

私も結構久しぶりっていうのもあったんだけどね......
